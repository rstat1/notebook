API Tokens

Tokens created here, may be used to access the DevCentral API with the same privileges as the creator. You'll need one of these if you want to use the dc CLI app.

Add a new API Token

Scopes

api

Grants standard non-administrative access to the API, including all groups and projects.

admin_api

Grants access to administrative portions of the API. Please note this scope does not include standard API access.

read_repo

Grants clone access to repos you have access to.

write_repo

Grants commit access to repos you have access to.
